# Plane

An extremely modular Discord bot designed to help RAF students with their RAFK (RAF Knowledge) and aircraft recognition.

## Invite

If you want to invite the bot hosted by Planet Waffle, you can do so [here](https://discord.com/api/oauth2/authorize?client_id=1044710491110522940&permissions=326417599488&scope=bot). This bot will always use the latest version from the main branch (I update the bot every time I merge a PR).
